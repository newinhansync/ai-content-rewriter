현재 개발된 Wordpress Plugin의 경우 서버 환경에서 인공지능을통해 답을 받아야하는 환경을 만들어야 하기에 몇가지 문제가 있다. 
1. 워드프레스 호스팅 환경에서 적절치 않음 -> 서버 환경을 컨트롤 못하는 상황에서 호스팅만으로 운영해야하는 상황에서 해당 타임아웃의 시간을 매우 길게 설정할 수 없음 
2. 자동화 어려움 -> 운영을 하면서 최종 모형은 자동화하는 모델을 만들고자 함. -> 이 또한 서버 호스팅 환경에서 적절치가 않음. 


1, 2번을 해결 할 수 있는 개발적인 방법을 찾아야 함. 

---

RSS로 추가한 게시글을 자동으로 포스팅 하게 하는 완전환 자동화 애플리케이션이 필요함. 
완벽자동화라는 것이 사람이 관여하여 큐레이팅 혹은 에그리에이션을 하지 않고도 인공지능이 이 블로그에 적절하게 포스팅 하는 것을 목표함. 포스팅 한 후 이미지 생성까지 완수하는 것을 목표함. 향후 추가 적인 파이프라인이 만들어 질 수도 있음. (해당 콘텐츠를 바탕으로 다양한 포맷 변화)
